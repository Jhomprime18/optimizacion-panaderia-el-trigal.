# Optimización de la Producción Semanal - Panadería El Trigal

Este repositorio contiene el script ejecutable desarrollado en Python puro para resolver el problema de mezcla de productos (*Product Mix*) utilizando Programación Lineal y Entera Mixta (MILP).

## 🛠️ Requisitos e Instalación
Para ejecutar el script, asegúrese de tener Python instalado y ejecutar el siguiente comando en la terminal de VS Code para instalar las dependencias:

```bash
pip install -r requirements.txt
